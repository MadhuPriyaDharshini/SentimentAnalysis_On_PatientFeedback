Title: Patient Pulse – A Pharmaceutical Analytics Portal

What's this Web app is about? 

Patient Pulse is a web application created using Streamlit,to transform fragmented,
 multi-language patient feedback into real-time, actionable drug performance insights (Drug’s Efficacy, Adverse events),
 to rapidly identify safety signals and optimize product strategies across global markets with the use of Snowflake’s Cortex and AI capabilities.
 
Technologies Used:
* Snowflake
* Streamlit
* Snowflake's Cortex AI capabilities
* Python for Data cleaning & analysis
* SQL
